# Alke Wallet

Estructura simple para proyecto de billetera virtual.

## Estructura
- `index.html`: Redirección
- `login.html`: Inicio de sesión
- `menu.html`: Dashboard principal

- `sendmoney.html`: Transferencias
- `transactions.html`: Historial
