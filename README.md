# Alke Wallet

Estructura simple para proyecto de billetera virtual.

## Estructura
- `index.html`: Redirección.
- `login.html`: Inicio de sesión.
- `menu.html`: Dashboard principal.
- `deposit.html`: Carga de saldo.
- `sendmoney.html`: Transferencias.
- `transactions.html`: Historial.

## Assets
- `assets/css`: Estilos base.
- `assets/js`: Lógica modules (config, auth, wallet, etc.).
- `assets/libs`: Dependencias (jQuery, Bootstrap).

## Funcionalidad
Todo estado se maneja en `localStorage` a través de `config.js`.
